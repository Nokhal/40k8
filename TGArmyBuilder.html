/tg/ 40kg Army Builder

Last updated : 2019 02 23

The goal of the army builder is simple :
:: Allowing you to quickly build a battle-forged matched play army using the latest 40k rules and sharing it easily. ::

Due to GW cavalier attitude to rules writing, there will always be leeways between RAI and RAW : complete rule checking from within an automated tool is impossible if the automated tool itself is not the authority on what is legal or not.

Henceforth, the objective of this tool is not to enforce all the rule for the players, but instead to present all the relevant rules directly in lieu of having to flip through hundreds of pages of maybe outdated informations.

This tool was built because battlescribe fail at every corner :
-The text export format is shit
-Due to the amateurism of it's contributors and ego wars, the catalogs are riddled with bugs and incorrect informations
-No information on one of the most important part of the game : stratagems
-Barely any rules are properly sourced

To save the army builder, simply save the page. (ctrl+s). It doesn't need internet to work.
To export your roster, simply copy paste the text to somewhere (notes, google doc, whatever)
To import a roster, paste it into the import dialog
